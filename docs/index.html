Projet mIRAcle

mIRAcle

Application Flask de consultation et d'entraînement autour de fiches de poste, quiz, ministères et statistiques.

GitHub Pages ne peut pas exécuter le backend Python de cette application. Cette page sert donc de vitrine statique. Pour la version fonctionnelle, il faut déployer l'app sur un hébergeur Python.

Pourquoi le 404

Ce que GitHub Pages attend

GitHub Pages publie des fichiers statiques. Il faut un fichier d'entrée comme index.html à la racine de la source publiée.

Ton projet principal utilise Flask, des routes dynamiques et gunicorn. Cette partie ne peut pas tourner sur GitHub Pages.

Déploiement réel

Pour faire tourner l'application

  • Utiliser Clever Cloud, Render, Railway ou un VPS.
  • Le projet est déjà préparé avec Procfile et requirements.txt.
  • Commande de démarrage: gunicorn app:app.
Usage recommandé

Organisation simple

  • GitHub Pages pour la vitrine ou la documentation.
  • Hébergeur Python pour l'application Flask.
  • Lien depuis cette page vers l'instance déployée quand elle existe.
